So fügen Sie Ihrer Webseite eine Cookie-Warnung hinzu

HTML für Cookie-Warnung

Das ist das benötigte HTML, um die Cookie-Warnung hinzuzufügen.

      
        <div id="cookie-warnung">
          <p>Wir verwenden Cookies, um Ihre Erfahrung zu verbessern...</p>
          <button onclick="akzeptiereCookies()">Akzeptieren</button>
        </div>

JavaScript für Cookie-Warnung

Dieser JavaScript-Code lässt die Warnung verschwinden, wenn Benutzer Cookies akzeptieren.

      
        <script>
          if (localStorage.getItem('cookiesAkzeptiert') === 'true') {
            document.getElementById('cookie-warnung').style.display = 'none';
          }
          function akzeptiereCookies() {
            localStorage.setItem('cookiesAkzeptiert', 'true');
            document.getElementById('cookie-warnung').style.display = 'none';
          }
        </script>

Wir verwenden Cookies, um Ihre Erfahrung zu verbessern. Durch die Nutzung dieser Website stimmen Sie unserer Cookie-Richtlinie zu.
